> A JavaScript util collections built for homedo.com


## Install

[npm][]:

```sh
npm install homedo-utils
```

[yarn][]:

```sh
yarn add homedo-utils
```


## Usage

```js
const HmdUtils = require('homedo-utils');

const hmdUtils = new HmdUtils();

console.log(hmdUtils.renderName());
// script
```
